# Business Analytics Platform

A comprehensive web application for business analytics and product management.

## Features

1. **Login Page** - Secure entry point to the application
2. **Business Type Selection** - Choose from retail, product-based, service-based, or custom business types
3. **Product Management** - Add, edit, and delete products with cost and pricing information
4. **Sales Tracking** - Record daily sales and advertising costs
5. **Analytics Dashboard** - Visualize sales data with charts and graphs
6. **AI Recommendations** - Get intelligent suggestions for product improvements

## Installation

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the development server

## Usage

1. Login with any username/password (demo credentials)
2. Select your business type
3. Add products with cost, price, and sales information
4. View analytics dashboard for insights
5. Follow AI recommendations to improve product performance

## Technologies Used

- React.js
- React Router
- Recharts for data visualization
- CSS3 for styling
- LocalStorage for data persistence
